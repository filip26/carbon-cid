### Building

Fork and clone the project repository.

#### Java 11
```bash
> cd carbon-decentralized-identifiers
> mvn clean package
```

#### Java 8
```bash
> cd carbon-decentralized-identifiers
> mvn -f pom_jre8.xml clean package
```
